#### ✅ 飞书风文字提取
> **`mini_agent.py`**｜不到 20 行  
> ```python
> import asyncio
> from claude_agent_sdk import query, ClaudeAgentOptions
> 
> async def main():
>     options = ClaudeAgentOptions(
>         max_turns=3,
>         allowed_tools=["Read", "Grep"],
>         system_prompt="You are a code reviewer.",
>     )
>     async for msg in query(
>         prompt="分析 src/api 模块",
>         options=options
>     ):
>         if msg.type == "assistant":
>             print(msg.content, end="", flush=True)
>         elif msg.type == "result":
>             print(f"\n成本: ${msg.cost} · 耗时: {msg.duration}s")
> 
> asyncio.run(main())
> ```
>   
> **📌 关键配置说明**  
> • `max_turns`｜限制对话轮次 · 防失控  
> • `allowed_tools`｜工具白名单 · 仅 Read / Grep  
> • `async for`｜流式遍历 · 实时打印 assistant
